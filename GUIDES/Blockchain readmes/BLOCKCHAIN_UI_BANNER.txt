┌──────────────────────────────────────────────────────────────────────────────┐
│                          TROUBLESHOOTING                                     │
└──────────────────────────────────────────────────────────────────────────────┘

  ISSUE: Status shows "Offline"
  → Check .env has ALGORAND_ADDRESS
  → Verify py-algorand-sdk is installed
  → Check network connectivity

  ISSUE: Balance is 0 ALGO
  → Visit https://bank.testnet.algorand.network
  → Request testnet ALGO (free)
  → Wait for confirmation (~5 seconds)

  ISSUE: TX shows "LOCAL-XXXX"
  → Connection failed, logged locally
  → All features still work normally
  → Check blockchain status panel

  ISSUE: No blockchain section
  → Old audit (before integration)
  → Submit new audit to see blockchain info
  → Hard refresh browser (Ctrl+F5)
